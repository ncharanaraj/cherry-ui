Alerts

An alert give a short, important message in a way that attracts the user's attention without interrupting the user's task.

Default Alerts

Default Alerts are available in 4 variants success alert, info alert, warning alert, error alert. To use this alert in your code, just include classname alert. And also as per the requirement you can also add alert-success, alert-info, alert-warning, alert-error as classnames. You can copy the html code from below code snippet.
